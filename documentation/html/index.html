Introduction

Analysis on Rasters is common in image processing and geographical analysis. This library provides tools to access geographical data sources as ranges. It provide wrappers and convenience functions to make use of any raster formats supported by GDAL. The different views of the raster allow iteration of pixels or edges (pairs of adjacent pixels) and in row-major or col-major direction. The library is work in progress, even though it is stabilising now I am using it for multiple projects. Please contact me (a.hagen-zanker@surrey.ac.uk) if you are interested in using this for your work or research.

Note: This is not a Boost library or intended to become one, I am just trying to use the tools recommended on the Boost Library Incubator.
